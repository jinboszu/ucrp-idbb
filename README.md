This project is part of the paper [Jin, B. & Tanaka, S. (2023). An exact algorithm for the unrestricted container relocation problem with new lower bounds and dominance rules. [*European Journal of Operational Research*](https://doi.org/10.1016/j.ejor.2022.04.006), 304(2), 494–514].

This project is written in C99.

Two versions are provided:
- `base`: Simplified version for tutorial
- `main`: Official version for research

**Remark on the name of the JZW heuristic:** The JZW heuristic was originally developed in Jin, Zhu, & Lim (2015), and should technically have been named as JZL based on the authors’ surnames’ initials. However, Tricoire, Scagnetti, & Beham (2018) and Feillet, Parragh, & Tricoire (2019) adopted the name JZW for the heuristic, with ‘J’ from the first author Bo Jin’s surname, and ‘Z’ and ‘W’ from the second author Wenbin Zhu’s surname and given name. In this paper, we adhere to using this naming convention to prevent confusion among readers.
